# SmithWaterman_algorithm
An algorithm of specifying similar sequence of nucleotides
## To learn this algorithm
visit https://en.wikipedia.org/wiki/Smith%E2%80%93Waterman_algorithm to learn how it works
## Compile 
1. type **make** in your terminal
2. type **./SW** in your terminal
3. input sequence A,B
4. wait for results(maybe many results)
